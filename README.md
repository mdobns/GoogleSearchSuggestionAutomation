# GoogleSearchSuggestionAutomation

---

This repo is for to automate google search from Excel file keys and store the suggested Largest and smallest search suggestion to the excel, It uses Selenium for webdriver and Python as Programming language.

## To run

### First install <i>Selenium</i>

```Python
pip install selenium
```

### Then Clone & run

```Python
git clone https://github.com/mdobns/GoogleSearchSuggestionAutomation
```

```Python
python Main.py
```
